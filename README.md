# QuantumGuard

QuantumGuard is a web-based Post-Quantum Code Security Scanner that helps identify common security vulnerabilities in source code while educating users about the impact of quantum computing on modern cryptography.

## Features

- 🔍 Scan source code for common security vulnerabilities
- 📊 Interactive dashboard and security reports
- 📄 Detailed scan results
- ⚛️ Quantum Computing education section
- 🔐 Quantum Random Number Generator (QRNG) simulation
- ⚡ Grover's Algorithm visualization
- 🔑 Shor's Algorithm visualization
- 🌙 Dark mode support

## Purpose

This project demonstrates how traditional cryptographic systems can be affected by future quantum computers while providing an educational platform to understand post-quantum cryptography.
